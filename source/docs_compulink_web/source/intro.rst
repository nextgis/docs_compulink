Введение
========

В настоящем документе представлено руководство пользователя Веб-ГИС **NextGIS Web Compulink** 
версии 2. В руководстве даны общие сведения о программном обеспечении 
(:abbr:`ПО (программное обеспечение)`), описание пользовательского и административного интерфейсов, 
рассмотрены базовые пользовательские задачи и задачи администрирования.
